BULLSASAUR

  1. First, draw circles for the head, body, eyes and legs
  2. Next, add a flower, back, and ears, then add outlines for the eyes and legs
  3. Next, remove all unnecessary lines.
  4. Next, add the mouth, nostrils, claws, outline pupils, and draw the spots.
  5. Finally, color spots on the body, pupils and claws.

CHARMANDER

MANKEY

  1. First, draw circles for the head, eyes, and all legs.
  2. Next, remove all unnecessary lines. Then add ears, pupils, paws, nose and tail.
  3. Next, remove all unnecessary lines, and add hair to the ears and body.
  4. Finally, remove all unnecessary lines, and add nostrils.
